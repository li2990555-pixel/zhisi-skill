# 人文学科标准论文结构模板

## 思辨型（哲学、理论美学、理论语言学）

```
第1章 绪论
  1.1 问题的提出（为何这个问题值得研究）
  1.2 研究现状与文献综述（前人的成果与不足）
  1.3 研究思路与方法（如何展开论证）
  1.4 核心概念界定（关键术语的定义）
  1.5 论文结构安排（各章概述）

第2章 概念谱系：核心术语的语义史
  2.1 概念的起源与演变
  2.2 不同传统中的理解差异
  2.3 本文的界定与操作化

第3章 理论框架：分析工具的建构
  3.1 理论渊源（继承了什么）
  3.2 理论创新（贡献了什么）
  3.3 分析框架的适用性论证

第4章 论证展开I：分论点A
  4.1 分论点A的前提与依据
  4.2 论证过程
  4.3 小结

第5章 论证展开II：分论点B（结构同上）

第6章 回应与反思
  6.1 可能的反驳与回应
  6.2 理论的限度与边界

第7章 结论
  7.1 研究结论
  7.2 理论贡献
  7.3 研究局限
  7.4 未来展望
```

## 历史型（文学史、艺术史、思想史）

```
第1章 绪论
  1.1 研究缘起
  1.2 文献综述
  1.3 研究方法（史料学方法、历史语境法）
  1.4 分期依据与结构说明

第2章 历史语境：时代背景与思想气候
  2.1 社会历史背景
  2.2 思想/文化气候
  2.3 知识谱系定位

第3章 史料考辨：关键文献的文本分析
  3.1 核心文献的版本与流传
  3.2 文献的真伪与年代考
  3.3 关键段落的解读

第4章 分期论述I：早期形态
  4.1 代表性人物/作品
  4.2 核心特征
  4.3 历史意义

第5章 分期论述II：发展演变（结构同上）

第6章 比较与综合
  6.1 历时比较（不同时期的演变线索）
  6.2 共时比较（同一时期不同流派）
  6.3 综合判断

第7章 结论
```

## 比较型（比较文学、跨文化研究）

```
第1章 绪论
  1.1 为什么比较？（可比性的论证）
  1.2 文献综述（已有比较研究的成果）
  1.3 比较的方法论反思
  1.4 比较维度的设定

第2章 比较对象A的独立分析
  2.1 A的核心特征
  2.2 A的内在逻辑
  2.3 A的发生语境

第3章 比较对象B的独立分析（结构同上）

第4章 比较框架
  4.1 可比性的再论证
  4.2 比较的具体维度
  4.3 比较的方法与步骤

第5章 异同分析
  5.1-5.3 各维度比较
  5.4 异同的深层原因分析

第6章 理论提升
  6.1 从个案到普遍：比较的启示
  6.2 对既有理论的修正或补充
  6.3 跨文化/跨语境的方法论贡献

第7章 结论
```

## 实证/案例型（语言学、传播学、艺术社会学）

```
第1章 绪论
  1.1 研究背景与问题
  1.2 文献综述
  1.3 研究问题与假设
  1.4 研究方法

第2章 理论框架
  2.1 核心理论
  2.2 分析框架
  2.3 操作化定义

第3章 研究设计
  3.1 数据来源
  3.2 样本选择
  3.3 分析方法

第4章 分析结果
  4.1 描述性结果
  4.2 主要发现
  4.3 稳健性检验（如适用）

第5章 讨论
  5.1 结果的理论解释
  5.2 与既有研究的对话
  5.3 研究局限

第6章 结论
```

## 各学科常见结构偏好速查

| 学科 | 偏好的结构类型 | 特别注意事项 |
|------|--------------|-------------|
| 哲学 | 思辨型 | 概念辨析先行，论证链条清晰 |
| 中国语言文学（古代文学） | 历史型/比较型 | 文献考据功底要展现 |
| 中国语言文学（文艺学） | 思辨型/比较型 | 理论深度是核心 |
| 艺术学（艺术史） | 历史型 | 图像/作品分析需图文对照 |
| 艺术学（艺术理论） | 思辨型 | 概念界定要严谨 |
| 历史学 | 历史型 | 史料批判意识必须展现 |
| 语言学 | 实证型/思辨型 | 语料/数据要有代表性 |
| 传播学 | 实证型/思辨型 | 方法论需明确 |
